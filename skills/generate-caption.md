# Skill: Generate Caption Media Sosial

## Tujuan
Membuat caption menarik untuk posting produk di Telegram, Facebook, dan Instagram.

## Format per Platform

### Telegram
```
🔥 [NAMA PRODUK]

⭐ Rating: X.X/5 | 🛒 Terjual: X.XXX+

✅ [Keunggulan 1]
✅ [Keunggulan 2]  
✅ [Keunggulan 3]

💰 Harga: Rp XX.XXX - Rp XX.XXX

👇 Beli Sekarang:
[AFFILIATE LINK]

#shopee #rekomendasiproduk #[kategori]
```

### Facebook
```
Lagi nyari [NAMA PRODUK]? 🤔

Ini dia yang paling banyak dibeli minggu ini!
Rating ⭐ [RATING]/5 dari [JUMLAH] pembeli

🎯 Cocok untuk: [target user]
💡 Keunggulan utama: [keunggulan]
💰 Harga terjangkau: mulai Rp [HARGA]

Link ada di komentar pertama 👇
(klik sebelum kehabisan stok!)

#rekomendasiproduk #shopee #[kategori]
```

### Instagram
```
[EMOJI] [NAMA PRODUK PENDEK] ✨

Sudah [JUMLAH_TERJUAL]+ orang beli ini!
Rating [RATING]/5 ⭐

Harga mulai Rp [HARGA] 💰
Link di bio ya! 

#shopee #[kategori] #rekomendasiproduk
#belanjashopee #produklokal #[keyword]
```

## Aturan Caption
- Jangan terlalu panjang untuk IG (max 300 karakter)
- Telegram boleh lebih panjang dan detail
- Selalu ada call to action
- Gunakan emoji yang relevan
- Hashtag 5-10 yang relevan
