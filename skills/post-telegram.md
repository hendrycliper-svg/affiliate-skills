# Skill: Post ke Telegram Channel

## Tujuan
Mengirim konten affiliate ke Telegram channel secara otomatis.

## Yang Dibutuhkan
- Telegram Bot Token (dari @BotFather)
- Channel ID atau username channel
- Bot sudah dijadikan admin di channel

## Cara Dapat Bot Token
1. Buka Telegram, cari @BotFather
2. Ketik /newbot
3. Ikuti instruksi, copy token yang diberikan

## Format Pesan
Gunakan skill generate-caption.md format Telegram.

## Jadwal Posting
- Pagi: 08.00 - 09.00 WIB
- Siang: 12.00 - 13.00 WIB  
- Malam: 19.00 - 20.00 WIB

## Catatan Penting
- Maksimal 3-5 post per hari agar tidak dianggap spam
- Selalu variasikan produk yang dipost
- Jangan post produk yang sama dalam 7 hari
- Gunakan gambar produk dari Shopee jika memungkinkan
