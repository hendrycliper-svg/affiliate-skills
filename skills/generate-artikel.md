# Skill: Generate Artikel Review SEO

## Tujuan
Membuat artikel review produk yang SEO-friendly untuk blog affiliate.

## Struktur Artikel Wajib
1. **Judul H1** - mengandung nama produk + keyword (max 60 karakter)
2. **Intro** - 2-3 paragraf hook yang menarik pembaca
3. **H2: Spesifikasi & Keunggulan** - detail produk
4. **H2: Kelebihan dan Kekurangan** - pros ✅ cons ❌
5. **H2: Kenapa Layak Dibeli** - paragraf persuasif
6. **H2: Harga dan Cara Beli** - info harga + tombol CTA
7. **H2: FAQ** - 3-4 pertanyaan umum
8. **Penutup** - CTA terakhir

## Format CTA Button
```html
<a href="AFFILIATE_LINK" class="btn-affiliate" target="_blank" rel="nofollow">
🛒 Beli di Shopee - Cek Harga Terbaru
</a>
```

## Aturan Penulisan
- Bahasa Indonesia santai tapi profesional
- Minimal 800 kata
- Keyword utama muncul 3-5x secara natural
- Jangan terasa seperti tulisan AI
- Gunakan emoji secukupnya
- Format output: HTML

## SEO Checklist
- [ ] Title mengandung keyword utama
- [ ] Meta description max 155 karakter
- [ ] Ada internal heading H2 dan H3
- [ ] Keyword di paragraf pertama
- [ ] Call to action jelas
